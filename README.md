
# Finance Tracker API

REST API сервис для управления личными финансами. Приложение позволяет пользователям вести учет своих доходов и расходов, управлять мультивалютными кошельками и осуществлять переводы между ними.

## Основные возможности
* Регистрация и авторизация пользователей на основе JWT-токенов
* Создание кошельков в различных валютах (RUB, USD, EUR)
* Учет операций: доходы, расходы и переводы между кошельками
* Автоматическая конвертация валют при переводах и подсчете общего баланса с использованием внешнего API курсов валют

## Требования к окружению
Для запуска проекта вам понадобятся:
* **Python** версии 3.12 или выше
* **PostgreSQL** (установленный локально или на удаленном сервере)
* **Docker Compose**

## Подготовка к запуску

### Настройка переменных окружения
В корневой директории проекта создайте файл `.env`. Приложение использует библиотеку pydantic-settings для чтения конфигурации 

Добавьте в файл `.env` следующие переменные, заменив значения на актуальные:

```env
# Настройки безопасности
SECRET_KEY=your_super_secret_key_here
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30

# Настройки подключения к PostgreSQL
POSTGRES_USER=postgres
POSTGRES_PASSWORD=your_db_password
POSTGRES_SERVER=192.168.0.222  
POSTGRES_PORT=5432
POSTGRES_DB=finance_db
```

### Настройка базы данных PostgreSQL
Приложение использует PostgreSQL в качестве основной базы данных. Перед запуском приложения необходимо создать пустую базу данных на вашем сервере PostgreSQL. Название созданной базы данных должно совпадать с переменной POSTGRES_DB


## Инструкция по запуску

#### Запуск через Docker Compose

Для сборки и запуска контейнеров в фоновом режиме выполните команду:
```bash
docker-compose up --build -d
```
После запуска сервера доступны следующие адреса:

* Веб-интерфейс: `http://127.0.0.1:8000/static/index.html `

* Swagger: `http://127.0.0.1:8000/docs`
